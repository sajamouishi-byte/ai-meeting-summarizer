#  AI Meeting Summarizer

A Python-based tool that uses AI to summarize meeting notes, extract decisions, and generate clear action points.

---

##  Features

-  Summarizes meeting notes into key points
-  Extracts decisions made during the meeting
-  Generates structured action points
-  Fast and simple command-line tool

---

## 🛠 Tech Stack

- Python
- OpenAI API
- dotenv (for secure API key handling)
